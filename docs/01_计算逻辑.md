# 📄 任务详情: 01_核心功能规格计算 (Functional Specs)

**核心指标 (严格按照补充 specs):**

### 1. Module A: 时光回溯
*   **全局起点**: 搜索全局 `msg_id` 最小的记录。
*   **年度首条**: 搜索 2025-01-01 后首条消息及其下一条回复（构建对话感）。

### 2. Module B: 数字化重量
*   **计算公式**: 
    *   照片 = MB / 5
    *   音乐 = MB / 4
    *   电影 = MB / 2500
*   *注意*: 需递归扫描 `dxa🥰_files` 目录。

### 3. Module C: 年度热力图
*   **Bucket 策略**: 按 `YYYY-MM-DD` 聚合消息总数。
*   **极值提取**: 寻找 `Max(count)` 的日期。

### 4. Module D: 习惯雷达
*   **分类标准**:
    *   `1` -> Text
    *   `image` -> Image
    *   `video` -> Video
    *   `47` -> Sticker
    *   `49` -> File/Link

### 5. Module E: 总览与最爱表情
*   **字数统计**: 仅针对 Text 类型进行 `len()` 累加。
*   **表情王**: 对 `media_path` 进行频率统计，取 Top 1 的资源链接。
